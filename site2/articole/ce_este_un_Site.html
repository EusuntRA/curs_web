Ce este un Site?

Noțiunea site web provine din expresia engleză web site și desemnează o grupă de pagini web multimedia (conținând texte, imagini fixe, animații ș.a.), accesibile în Internet în principiu orișicui, de obicei pe o temă anume, și care sunt conectate între ele prin așa-numite hiperlinkuri. Diversele site-uri web pot fi create de către o organizație, o persoană particulară, instituții publice etc. Inițial noțiunea apărea în limba română scrisă sub două forme : sit web și site (ca în limba engleză).

Intrebarea 1: Am nevoie de un site sau de un blog?
Ei bine, un site este o colectie de pagini web care sunt interconectate. Poate sa cuprinda o pagina principala (home page), landing page, pagina de vanzari, blog, articole, forum, dar nu neaparat. Un blog este aproape ca un jurnal online. Poate sa fie plasat pe un server dedicat sau poate sa fie stocat pe o platforma de alte bloggere. Poate ca un blog poate fi stocat pe o platforma comuna, insa niciodata un site de prezentare al firmei, de exemplu, deoarece nu detii controlul exclusiv asupra acestuia si poti pierde accesul total in masura in care primesti ban de la administrator.

In ziua de astazi, foarte multe site-uri de prezentare contin un blog deoarece informatia poate fi usor actualizata. Este un instrument care te ajuta sa iti promovezi mai usor produsul sau serviciul. Totodata, blogurile acorda posibilitatea de adaugare comentarii pentru cresterea interactivitatii cu site-ul.

Intrebarea 2: Ce este un landing page?
Este acea pagina pe care un vizitator aterizeaza in urma unui proces de cautare al unor cuvinte cheie. Un site poate avea mai multe landing pages intrucat, fiecare pagina este optimizata - sau ar trebui sa fie - pentru anumite cuvinte cheie pe baza carora se pot executa cautari automate. Astfel, cand utilizatorul introduce anumite cuvinte cheie in motorul de cautare, acesta afiseaza paginile relevante pentru acele cuvinte.

Astfel, se evita frustrarea care apare atunci cand tu cauti ceva anume prin cuvinte cheie, numai ca sa fii condus pe o pagina principala generica din care incerci sa iti dai seama sa ajungi pe o alta pagina unde se gaseste informatia de care chiar ai nevoie. Orice pagina din site care este optimizata pentru anumite cuvinte cheie poate deveni un landing page.

Intrebarea 3: Ce este Web 2.0?
Web 2.0 este termenul care descrie a doua generatie de World Wide Web care este focalizat pe abilitatea utilizatorilor de a colabora si de a contribui cu informatii online. Blogurile, serviciile web, retelele sociale sunt toate vazute ca facand parte din Web 2.0.

Dar restul?
Home page este pagina care este afisata atunci cand tastezi numele domeniului in bara de adresa.
Pagina de vanzari este realizata pentru vanzarea de produse si servicii.
Un articol este format din informatie care explica o anumita notiune.
Un post este un continut generat dinamic. Poate sa apara pe un blog sau pe un forum.
Un thread este un topic deschis pe un forum.
Un forum este o interfata unde diferiti utilizatori interactioneaza pe anumite topicuri.
