Ce este un Site?

Noțiunea site web provine din expresia engleză web site și desemnează o grupă de pagini web multimedia (conținând texte, imagini fixe, animații ș.a.), accesibile în Internet în principiu orișicui, de obicei pe o temă anume, și care sunt conectate între ele prin așa-numite hiperlinkuri. Diversele site-uri web pot fi create de către o organizație, o persoană particulară, instituții publice etc. Inițial noțiunea apărea în limba română scrisă sub două forme : sit web și site (ca în limba engleză).
